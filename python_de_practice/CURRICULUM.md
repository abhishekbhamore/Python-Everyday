# Python for AWS Data Engineering - Complete Learning Path

## Overview
This curriculum takes you from Python basics to AWS data engineering. Each day builds on previous knowledge with hands-on exercises.

## Learning Path

### Day 01: Python Fundamentals & Data Types
**Goal**: Master basic Python syntax and data types
- Variables and naming conventions
- Data types: int, float, str, bool
- Type casting and conversions
- String manipulation and f-strings
- Basic arithmetic and logical operations
- Print statements and input

### Day 02: Data Structures
**Goal**: Work with collections that power data pipelines
- Lists: creation, indexing, slicing, methods
- Dictionaries: key-value pairs, access, modification
- Tuples: immutability and use cases
- Sets: unique values and operations
- List/Dict comprehensions

### Day 03: Control Flow & Functions
**Goal**: Write reusable, logic-based code
- If/elif/else statements
- For and while loops
- Function definition and parameters
- Return values and scope
- Lambda functions
- Decorators basics

### Day 04: File I/O & Data Processing
**Goal**: Read, write, and manipulate data files
- Opening/closing files
- Reading/writing text files
- Context managers (with statement)
- File operations (append, update, delete)
- Parsing and processing file content

### Day 05: Working with JSON & CSV Data
**Goal**: Handle the most common data formats
- JSON parsing and creation
- CSV reading/writing
- Data validation
- Handling missing/corrupt data
- Data transformation pipelines

### Day 06: Error Handling & Data Validation
**Goal**: Build robust, production-ready code
- Try/except/finally blocks
- Custom exceptions
- Input validation
- Data quality checks
- Logging

### Day 07: Introduction to AWS Libraries
**Goal**: Get started with AWS SDK (boto3)
- Introduction to boto3
- S3 file operations (read/write)
- Basic DynamoDB operations
- Environment variables and credentials
- Error handling for AWS operations

### Day 08: Working with DataFrames (Pandas)
**Goal**: Master the main data processing library
- Creating DataFrames
- Data cleaning and transformation
- Grouping and aggregation
- Merging and joining data
- Exporting data

### Day 09: APIs & Data Retrieval
**Goal**: Fetch data from external sources
- HTTP requests basics
- REST API consumption
- Authentication (API keys)
- Error handling for API calls
- Data extraction and parsing

### Day 10: Real-world Data Pipeline Project
**Goal**: Build a complete data engineering pipeline
- Fetch data from API
- Validate and transform
- Load to S3 or database
- Error handling and logging
- Documentation

---

## How to Use This Curriculum
1. Each day has `exercises.py` (your code) and `test.py` (examples to run)
2. Read the exercise requirements carefully
3. Write code in `exercises.py`
4. Run `test.py` to see expected output
5. Compare your approach with solutions
6. Don't move to next day until you understand current day

## AWS Data Engineering Context
Throughout this course, you'll learn Python in the context of:
- Data ingestion from multiple sources
- Data transformation and cleaning
- Error handling for production systems
- Working with cloud storage (S3)
- Integration with AWS services
- Building reusable, maintainable pipelines
